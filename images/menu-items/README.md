# Menu Items Picture Folder

This folder is for storing pictures of all menu items. Add your images here and they will automatically appear in the menu section.

## Supported Formats
- JPG/JPEG
- PNG
- WebP
- GIF

## Image Recommendations
- **Minimum size:** 280x220 pixels
- **Recommended size:** 600x400 pixels or higher
- **Aspect ratio:** Any (will be cropped to fit)
- **Format:** JPG or PNG for best compatibility

## Steps to Add Images
1. Take or prepare photos of your menu items
2. Name them according to the naming convention above
3. Place them in this `menu-items` folder
4. Images will automatically appear in the menu section on your website

## Example
If you want to add a picture for "Fuchka w/ Chotpoti":
1. Take a photo of the dish
2. Name it `fuchka.jpg`
3. Place it in this folder: `images/menu-items/fuchka.jpg`
4. Refresh your website and it will appear!

---
**Note:** The website currently uses emoji placeholders. When you add images following this naming convention, you can update the HTML to use them instead of placeholders.
